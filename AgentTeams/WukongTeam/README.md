# Wukong Team

## 定位

Wukong Team 面向探索性、高不确定性、长周期任务。它不是普通 Team 的皮肤，而是一种更强调破局、借力与复盘节奏的 Team 形态。

## Leader

- `Sun Wukong`：负责快速探路、突破阻塞，并带领团队推进复杂探索任务。

## 成员

- `Tang Sanzang`：负责锚定使命与长期方向。
- `Zhu Bajie`：负责提出现实约束与权衡压力。
- `Sha Wujing`：负责稳定执行与持续推进。
- `White Dragon Horse`：负责承载上下文并维持连续性。

## 内建 Workflow

`make-vow -> explore -> borrow-strength -> break-through -> clear-gate -> review`

## 设计说明

- 适用于高不确定性任务，不应滥用于简单直线型任务。
- 保留人格化表达，但仍要求明确职责边界与交付标准。
- 强调阶段推进与复盘，而不是一次性线性执行。
