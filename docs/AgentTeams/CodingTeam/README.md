# Coding Team

## Positioning

Coding Team handles code implementation, modification, debugging, verification, and delivery.

## Leader

- `Tech Lead`: receives the task, delegates by default, and converges the final result.

## Members

- `Planner`: breaks work into a lightweight coding plan.
- `Scout`: locates code entry points and implementation clues.
- `Builder`: applies code changes.
- `Reviewer`: checks solution quality and consistency.
- `Verifier`: validates the result with tests or diagnostics.

## Built-in Workflow

`intake -> locate -> plan-or-delegate -> implement -> verify -> summarize`

## Design Notes

- Prefer the smallest coordination needed for the task.
- Keep verification in the default path.
- Do not force heavy collaboration on simple coding work.
