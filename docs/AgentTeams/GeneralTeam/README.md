# General Team

## Positioning

General Team handles research, analysis, writing, planning, decision support, and general operational work.

## Leader

- `Task Lead`: receives the task, delegates by default, and consolidates the final result.

## Members

- `Researcher`: collects information and source material.
- `Analyst`: structures comparisons and reasoning.
- `Writer`: produces the main draft.
- `Editor`: compresses and polishes the output.
- `Operator`: advances operational steps and task checklists.

## Built-in Workflow

`intake -> clarify -> research-or-analyze -> draft-or-execute -> check -> summarize`

## Design Notes

- Avoid coding-specific ceremony.
- Optimize for clarity, synthesis, and useful execution guidance.
