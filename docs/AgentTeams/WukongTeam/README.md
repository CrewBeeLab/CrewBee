# Wukong Team

## Positioning

Wukong Team handles exploratory, uncertain, and long-cycle tasks that benefit from staged breakthroughs and stronger cultural flavor.

## Leader

- `Sun Wukong`: opens paths quickly, breaks blockers, and leads difficult exploratory work.

## Members

- `Tang Sanzang`: preserves mission intent and direction.
- `Zhu Bajie`: pressures plans with practicality and tradeoffs.
- `Sha Wujing`: provides stability and reliable follow-through.
- `White Dragon Horse`: carries context and keeps the journey moving.

## Built-in Workflow

`make-vow -> explore -> borrow-strength -> break-through -> clear-gate -> review`

## Design Notes

- Treat the team as a distinct exploratory mode, not a cosmetic skin.
- Use it for difficult, uncertain work where iteration and reframing matter.
