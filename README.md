# AgentScroll

AgentScroll is a portable Agent Pack system for OpenCode, Codex, ClaudeCode, and other hosts.

## What this repository contains

- `src/core`: core module directory.
- `src/packs`: pack module directory.
- `src/adapters`: adapter module directory.
- `src/manager`: manager module directory.
- `docs`: architecture and planning notes.

## Initial scope

This is a minimal scaffold. The four modules are simple directories, not standalone npm packages.

## Quick start

```bash
npm install
npm run typecheck
```

## Note

No data model definitions and no feature implementations are included in this scaffold.
